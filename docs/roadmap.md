# The Nexus Chronicles — Roadmap

## Vision
A deep, expansive sci-fi exploration RPG set in a fractured galaxy, inspired by Isaac Asimov, Arthur C. Clarke, Frank Herbert, and Philip K. Dick. Players captain a starship, trading goods, exploring star systems, engaging in tactical combat, and uncovering an ancient mystery.

## Phase 1: Core Foundation ✅
- [x] Project scaffolding (Vite + React + TypeScript)
- [x] MobX state management
- [x] Three-layer architecture (UI → Store → Service)

## Phase 2: Core Game Systems
- [ ] Galaxy generation (30+ star systems, connections, factions)
- [ ] Player & ship management
- [ ] Dynamic economy (12+ commodities, supply/demand)
- [ ] Turn-based tactical combat
- [ ] Story engine (branching narrative, 5 chapters)
- [ ] Save/load system (IndexedDB + localStorage)
- [ ] NPC AI (traders, faction ships)
